Using it

Build once (while online):

docker build -t gothoom-devkit:latest .


Run interactive dev shell:

docker run --rm -it gothoom-devkit:latest
# inside
rebuild
ls -lh /binaries


Get artifacts out:

cid=$(docker create gothoom-devkit:latest)
docker cp "$cid":/binaries ./out
docker rm "$cid"


Back up the entire dev kit image (toolchains + caches + source + binaries):

docker save -o gothoom-devkit_full.tar gothoom-devkit:latest
# later/elsewhere
docker load -i gothoom-devkit_full.tar
